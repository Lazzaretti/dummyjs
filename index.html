Dummy Text
Use the attribute "data-dummy" for dummy text
<p data-dummy></p>

Specify an amount of dummy words
<em data-dummy="20"></em>
Random amount of dummy words
<div data-dummy="3,6"></div>
Dummy Images
Use the attribute "data-dummy" for a dummy image
Auto Size to parent container
<img data-dummy />
Specific Size
<img data-dummy="400x300" />
or,
<img data-dummy width="400" height="300" />
or, you can also size with CSS
Custom Text
<img data-dummy data-text="Custom Text That is really long" />
Custom Colors
<img data-dummy data-color="#0000ff" data-text-color="#fff" />

Backstory

In a continuous effort to streamline front end builds I found myelf creating the front end through different phases. First layout, complex elements, then finaly integration of assets and real content before CMS/framework integration.

It was these early stages I was continously copy pasting elements to form, then repeating that process everytime a change was made. Testing dynamic content & word wrapping I was entering fake text data of differnet lenths over and over to test. As all developers do, this was finally automated into this small framework with an interface that aids the build process.
